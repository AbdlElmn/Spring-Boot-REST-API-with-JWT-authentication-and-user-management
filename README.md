# Spring Boot User Auth (JWT)

A simple Spring Boot REST API that implements user authentication using JWT (JSON Web Token).

## Features

* User registration
* User login
* JWT-based authentication
* Password hashing using BCrypt
* Protected endpoints

## Tech Stack

* Java 21
* Spring Boot
* Spring Security
* JWT (io.jsonwebtoken)
* Maven

## Notes

* `application.properties` is excluded for security reasons
* Use your own database configuration and JWT secret

## How to Run

1. Clone the repository
2. Configure your `application.properties`
3. Run the application
